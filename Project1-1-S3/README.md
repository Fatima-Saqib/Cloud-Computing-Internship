# Project 1 - Static Website Hosting using Amazon S3

## Objective
Host a static website using Amazon S3.

## Services Used
- Amazon S3

## Project Description
This project demonstrates how to deploy a static website using Amazon S3. An S3 bucket was created, website files were uploaded, static website hosting was enabled, and the website was made publicly accessible.

## Technologies
- AWS S3
- HTML
- CSS
